# 投票应用

## 架构

```
  .  
  ├── healthchecks--------------------（检查数据库健康状态）  
  ├── result--------------------------（管理员查询组件）  
  ├── vote---------------------------- (用户投票组件) 
  └── worker--------------------------（后端工作程序）
```

## 运行

- 操作系统: Ubuntu 22.04LTS
- docker-engine: 27.2.0

在项目根目录下运行指令`docker-compose up`即可,访问宿主机端口`localhost:8080`为投票界面,访问宿主机端口`localhost:8081`为查询界面
